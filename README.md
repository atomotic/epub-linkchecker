# epub-linkchecker

extract and check links from an epub file

## installation

Download a [binary](https://github.com/atomotic/epub-linkchecker/releases) or use `go get`:

	go get -u -v github.com/atomotic/epub-linkchecker
	
## usage

	$ epub-linkchecker {file.epub}
